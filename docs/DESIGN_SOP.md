# Team Alpha Photography - Website Design SOP

This SOP defines how interns should design, develop, and deliver the Team Alpha Photography website. The goal is to create a premium wedding photography website inspired by the Pic-Time aesthetic: clean, emotional, minimal, and conversion-focused.

## 1. Goal
To create a premium wedding photography website inspired by the Pic-Time aesthetic: clean, emotional, minimal, and conversion-focused.

## 2. Brand Overview
*   **Brand Name**: Team Alpha Photography
*   **Industry**: Wedding Photography
*   **Brand Personality**: Elegant • Emotional • Minimal • Premium • Trustworthy

**Our website should:**
*   Highlight emotions & storytelling
*   Feel luxurious, not crowded
*   Build trust with couples & families
*   Convert visitors into inquiries

## 3. Design Reference & Inspiration
*   **Primary inspiration**: Pic-Time (www.pic-time.com)
*   **Visual Direction**:
    *   Soft, neutral tones
    *   Large image blocks
    *   Clean typography
    *   High white-space usage
    *   Minimal animations

**Important**: We are inspired by the style, not copying layouts pixel-by-pixel.

## 4. Color Palette (Mandatory)
Use a soft, neutral, luxury palette similar to Pic-Time:

**Primary Colors**
*   **Ivory / Off-white**: `#F7F5F2` (or similar)
*   **Charcoal Black**: `#1C1C1C` (or similar)
*   **Warm Grey**: `#B8B5B0` (or similar)

**Accent Colors**
*   Soft beige
*   Muted brown (`#8C7B6D`)
*   Very subtle gold tones (`#D4AF37`) (optional, sparingly)

**❌ Avoid:**
*   Bright colors
*   Neon shades
*   Strong contrasts

## 5. Typography fonts (Mandatory)
**Headings (H1–H3)**
*   Elegant serif font (similar to Pic-Time)
*   Examples: Playfair Display, Cormorant, Libre Baskerville

**Body Text**
*   Clean sans-serif
*   Examples: Inter, Lato, Helvetica / system fonts

**Rules**
*   Do not use more than 2 font families
*   Line spacing must feel airy and readable
